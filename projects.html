Yah-du-ot

February-May 2017

GUI based dice game written in Java utilizing the Java Foundation Classes. Created in a three person team as an object-oriented-programming class project. Planned first using UML class and sequence diagrams and following the model-view-controller design pattern. The project focused on UI and event based programming object-oriented-design, and professional group work. The game itself is a mix between Yahtzee, Sudoku, and Dots & Boxes.

  • Source control with Git and GitHub
  • Documentation with Javadoc
  • Testing with JUnit
  • User acceptance testing
